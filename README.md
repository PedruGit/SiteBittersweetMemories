# Portal Bittersweet Memories

> Um documento web feito em homenagem a um dos meus álbuns de música favoritos, criado pelo artista Yun Li.

Este documento foi desenvolvido para praticar meus estudos de HTML e principalmente CSS.

---

## Instalação e Execução

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

[Visual Studio Code](https://code.visualstudio.com/download?_exp_download=fb315fc982)

### Passo a Passo
1. **Clone o repositório:**

```bash
   git clone https://github.com/PedruGit/AplicativoQuizHarryPotter.git
```

2. Abra a pasta e execute o arquivo com ícone do seu navegador.
